# Web系企業一覧<!-- omit in toc -->
日本のWeb系の企業一覧(一部)。

一応ゲーム系/組み込み系/その他もWeb系を扱う場合、一部含める。

外資の日本支部(傘下)もここに含める。

日本のメガベンチャーは上場済み andor 資本金が5億以上 andor または従業員数1000人以上。

スタートアップは創業5年以内(2018年以降設立)のWeb系企業と定義。

それ以外は その他Web系企業 として扱う。
